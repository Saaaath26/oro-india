📧 Email Service (SMTP)

Disabled

Email service is currently using development mode (OTP shown in browser).

To enable real email sending:

1. Get Gmail App Password or configure SMTP service
2. Update .env file:
EMAIL_USER=your-email@gmail.com EMAIL_PASS=your-app-password
3. Restart the server

🔐 Google OAuth Authentication

Disabled

Google Sign-In is currently disabled and shows "Coming Soon" to users.

To enable Google OAuth:

1. Create Google Cloud Project
2. Enable Google+ API
3. Create OAuth 2.0 credentials
4. Update .env file:
GOOGLE_CLIENT_ID=your-google-client-id GOOGLE_CLIENT_SECRET=your-google-client-secret
5. Update frontend code (remove disabled attributes)
6. Restart the server
