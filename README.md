# EduMaster - CBSE Class 10 Preparation Platform

A comprehensive web-based learning platform designed specifically for CBSE Class 10 students preparing for Science and Social Science subjects.

## Features

### 🎯 5-Tier Difficulty System
- **Foundation**: Basic concepts and fundamental understanding
- **Core**: Core concepts with moderate complexity  
- **Advanced**: Advanced topics with analytical thinking
- **Expert**: Expert level with complex problem solving
- **Master**: Master level with comprehensive understanding

### 🧠 AI-Powered Learning
- Personalized chapter summaries generated dynamically
- Detailed explanations with key concepts and real-world applications
- Smart content adaptation based on user progress

### 📚 Comprehensive Content
**Science Chapters:**
- Light - Reflection and Refraction
- Carbon and its Compounds
- Life Processes
- Heredity and Evolution
- Management of Natural Resources

**Social Science Chapters:**
- The Rise of Nationalism in Europe
- Forest and Wildlife Resources
- Water Resources
- Agriculture
- Democratic Politics

### 🎮 Interactive Testing System
- Dynamic question pool with 500+ questions
- Non-repeating questions across attempts
- 15-minute timed tests
- Instant results and progress tracking
- 75% pass rate required for level progression

### 👥 Multi-User Support
- **Student**: Full access to learning materials and tests
- **Teacher**: Enhanced analytics and progress monitoring
- **Parent**: Progress overview and performance insights
- **Admin**: Complete platform management

### 📱 Modern UI/UX
- Beautiful gradient designs with smooth animations
- Fully responsive for all devices
- Intuitive navigation and user-friendly interface
- Progress visualization with charts and badges

## How to Use

1. **Open the Platform**: Open `index.html` in your web browser
2. **Select User Type**: Choose your role (Student/Teacher/Parent/Admin) from the dropdown
3. **Choose Subject**: Click on Science or Social Science
4. **Select Chapter**: Browse and select any chapter to start learning
5. **Read AI Summary**: Get personalized chapter summaries before testing
6. **Take Tests**: Start with Foundation level and progress through difficulty tiers
7. **Track Progress**: Monitor your advancement through the 5-tier system

## Progression System

- Start at **Foundation** level for any chapter
- Score 75%+ to unlock the next difficulty level
- Unlimited attempts with different questions each time
- Complete all 5 levels to master a chapter
- Progress is automatically saved in your browser

## Technical Details

- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+)
- **Storage**: Browser localStorage for progress tracking
- **Responsive**: Works on desktop, tablet, and mobile devices
- **No Backend Required**: Runs entirely in the browser
